# koeta-frontend

Frontend aplikasi **Koeta**.

## 🚀 Menjalankan Project

### 1. Install dependency
```bash
npm install
```

### 2. Jalankan mode development
```bash
npm run dev
```

### 3. Build untuk production
```bash
npm run build
```

### 4. Preview hasil build
```bash
npm run preview
```

## 🛠️ Tech Stack

- JavaScript / TypeScript
- Node.js + npm
- Framework frontend (sesuai konfigurasi project)

## 📁 Struktur Folder (umum)

```text
koeta-frontend/
├── public/
├── src/
├── package.json
└── README.MD
```

## 📌 Catatan

- Pastikan versi Node.js sesuai kebutuhan project.
- Jika ada file `.env`, salin dari `.env.example` lalu sesuaikan nilainya.
